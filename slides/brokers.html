Популярные брокеры

Apache ActiveMQ

  • Один из первых
  • Поддерживает все популярные протоколы
  • Интерес к проекту уже гаснет, если верить Google Trends
  • Однако, все еще является отличным швейцарским ножом, используется в энтерпрайзе в основном
  • Написан на Java

RabbitMQ

  • Самый популярный в текущее время
  • Работает из коробки, enjoy your messages
  • Расширяется плагинами
  • Даже через вебсокеты общаться можно
  • Написан на Erlang

ZeroMQ

  • Оверкилл для обычного применения
  • По сути свооей является фреймворком
  • Написан на C++
  • Мультиплатформенный, из-за этого пользовался в CERN
